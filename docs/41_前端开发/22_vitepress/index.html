简洁的内容创作

使用 Markdown 就能轻松编写美观、易读的文档内容，让创作者可以专注于核心表达，而不被复杂的格式要求干扰。

畅享 Vite 极速开发体验

借助 Vite 强大的即时编译和热更新能力，项目启动更迅速、修改反馈更及时，同时可以无缝集成丰富的插件生态。

自定义 Vue 组件扩展

支持在 Markdown 中使用 Vue 语法和组件，方便实现复杂的组件逻辑，极大提升了文档的灵活性和表现力。

🚀

极速加载与流畅页面切换

静态 HTML 实现了初次加载的极速体验，客户端路由则带来了导航切换的流畅感，让用户在浏览时几乎感受不到停滞。

🎨

丰富的主题定制支持

提供高度自由的主题配置，能够通过简单的配置或使用 Vue 组件，创造出独特风格的文档站点，更好地贴合品牌调性。

🔌

轻松集成第三方插件

轻松兼容 Vite 插件和其他 Vue 插件，不仅能扩展文档站点的功能，还能提升站点的交互性和用户体验。

🌐

完善的 SEO 和 PWA 支持

默认支持 SEO 优化设置和 PWA（渐进式网络应用），帮助站点获得更高的搜索引擎排名，同时提升离线访问体验。

🌍

多语言与国际化支持

内置多语言支持，能够轻松配置国际化，满足全球化项目需求，让文档更易于在全球范围内传播和使用。

📚

丰富的内置 Markdown 功能扩展

除了基本的 Markdown 语法外，还支持自定义容器、图表、代码块高亮等多种高级功能，满足各类内容创作需求。

🧩

现代化开发工具的友好集成

完美适配现代化的开发流程，支持 Git、CI/CD 和版本控制，让文档更新流程更加自动化、高效，适应团队协作需求。

🔍

强大的搜索功能

内置快速搜索功能，让用户能够迅速找到所需内容，提升用户体验，确保信息获取的高效与便利。

📈

实时统计与分析

提供访问数据的实时统计和分析，帮助站点管理员了解用户行为，从而做出更好的优化决策，提升用户满意度。
