# 娱乐社区

# 社区

  • Github ： 全球最大开源社区。
  • CSDN ：老牌中文技术博客社区。
  • 掘金 ： 一个帮助开发者成长的社区。
  • 简书 ：并不是纯粹的技术博客社区，很多伤春悲秋的东西。
  • 码云 ：可以看做GitHub的国内版，码云是个不错的替代者。
  • LeetCode ：经典的刷题网站，主要是算法题。
  • LintCode ：和LeetCode类似。
  • 牛客网 ：一个联网求职学习交流社区。
  • 思否 ：包含博客、问答的技术社区。
  • 开源中国 ：一个技术博客社区。
  • 51CTO ：一个IT技术网站。
  • V2EX ：页面风格很朴素，B格很高。
  • 腾讯云社区 ：腾讯云的开发者社区。
  • 阿里云社区 ：阿里云的开发者社区。
  • 开发者头条 ：一个程序员分享平台。
  • GitChat ：一个技术博客社区。
  • 知乎 ：知乎是个综合性的问答社区，但是聚集的程序员也比较多。
  • Stack Overflow ：全球最活跃的程序员技术问答交流社区。
  • http://dev.io ：不错的技术社区。
  • DZone ：世界上最大的在线社区之一。
  • Bytes ：一个面向开发人员和IT专业人员的交流社区。
  • Google Developers ：google开发社区。

# 在线教程

  • 菜鸟教程 ：在线教程网站，前端学习看这两个网站就够了。
  • 易百教程 ：内容比较全面的在线教程网站。
  • 码农教程 ：大量IT编程入门教程(JAVA, PHP, JAVASCRIPT, C, C++, HTML, CSS等)。
  • 简单教程 ：大量IT编程入门教程(前端、移动端、Java、.net等)。
  • Break易站 ：比较多的编程教程(Java、前端、服务端部署等)。
  • C语言中文网 ：以C语言为主，也包含数据结构、C++、Linux等。
  • http://jenkov.com ：一个国外的技术网站，提供了非常不错的Java教程。
  • http://baeldung.com ：也是一个国外的技术网站，提供了非常不错的Java教程。

# 视频教程

  • B站 ：众所周知，bilibili也可以用来学习，除了番剧、鬼畜之外，b站同样有非常多的高质量的编程教学视频。
  • 慕课网 ：比较不错的编程视频教学网站，可以找到比较体系的东西，当然，有些内容要花钱就是了。
  • 中国大学MOOC ：有很多名校的课程，当然不止CS。
  • 网易云课堂 ：有一些计算机编程方面的视频教程。
  • 实验楼 ：IT技术课程网站。
  • 大学生自学网 ：有一些计算机软件的大学课程。
  • 极客学院 ：IT编程课程网站，付费内容居多。
