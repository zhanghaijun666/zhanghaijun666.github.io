                            # kubectl

                            官网地址：https://kubernetes.io/zh-cn/docs/tasks/tools/install-kubectl-windows/open in new window

                              1. 下载地址：https://storage.googleapis.com/kubernetes-release/release/v1.9.0/bin/windows/amd64/kubectl.exeopen in new window
                              1. 设置环境变量 path
                              1. 配置

                              配置 kubernetes config 文件，这个文件是你在 linux 服务器上使用 kubectl 与 kubernetes 交互的配置文件

                              在 linux 上找到 HOME/.kube/config文件将其复制到windows的HOME/.kube/config 即可

                              我的位置是 C:/Users/admin/.kube/config
